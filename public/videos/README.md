# Bubu's Message Video Folder

Place your video message file here!

## Requirements:
- Filename: **`message.mp4`**
- Recommended Format: H.264 video with AAC audio (standard MP4)
- Both vertical (portrait 9:16) and landscape (16:9) formats are automatically supported!
- You can optionally place a poster image file here (e.g. `poster.jpg`) and edit the `VIDEO_POSTER` constant in `src/pages/MusicVideoTribute.tsx` to reference it.
